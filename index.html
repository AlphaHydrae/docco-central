docco-central

Docco documentation for whole projects.

Installation

npm install -g docco-central

Docco requires Pygments for syntax highlighting. Install it with sudo easy_install pygments or follow the installation documentation.

Usage

cd /path/to/your/project
docco-central lib/*.js

Documentation is put in docs by default.

Options

Usage: docco-central [options] <file>...

Options:

  -o, --output    change the output directory ("docs" by default)
  -q, --quiet     silence console logs
  -h, --help      show usage information
  -V, --version   show version number
